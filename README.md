# Simon Game 🎮

A classic memory-based Simon Game built using **HTML**, **CSS**, **JavaScript**, and **jQuery**. The game challenges players to remember and repeat an increasingly complex sequence of colors and sounds.

## Features

* Interactive gameplay based on memory and pattern recognition
* Dynamic color animations and visual feedback
* Keyboard-controlled game start
* Random sequence generation
* Progressive difficulty with increasing levels
* Responsive user interface

## Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
* jQuery

## How to Play

1. Press any key to start the game.
2. Watch the color sequence displayed by the game.
3. Repeat the sequence by clicking the colored buttons.
4. Each successful round adds a new color to the sequence.
5. The game ends if you click the wrong color.
